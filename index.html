Linguagem de Marcação de Hipertexto


Uma linguagem de marcação de hipertexto é uma linguagem utilizada para estruturar uma página, podendo adicionar formatação com tags.
O HTML5 é uma linguagem de marcação de hipertexto, e muitos o confundem com linguagem de programação, que tem outro proposito em um site.
O html em seu começo era estruturado com o uso de tabelas, hoje ele possui semantica para elaboração e estruturação da página, deixando as tabelas para exibição ou para estruturação de e-mails.

Acessibilidade com HTML5?


No HTML5 a acessibilidade é algo essencial em todo processo de desenvolvimento desde o uso de tags especificas de organização e estruturação.
O uso das tags tem o proposito de identificar o site e suas caracteristicas, apontando no código estruturas de cabeçalho, corpo do site e rodapé, como também identificar campos de artigos, imagens e multimidia.
No uso de mídia permite adicionar complementos de descritivos para melhor explicar um conteúdo visual a pessoas com deficiencia.

Tags principais utilizadas na estrutura para acessibilidade:


      <header> <nav> <main> <article> <footer>

Estrutura Semântica de HTML


A estrutura semântica do html garante que o código de seu site seja interpretado pelos buscadores web, fornecendo informações de acessibilidade, seguindo boas práticas de programação que garantem que o site atinja seu público alvo.

As tags traduzem aos motores de busca o que esta sendo exibido ao usuário, abaixo seu significado:

  1. Header - O cabeçalho da página.
  2. Nav - A navegação da página.
  3. Main - O conteúdo principal da página.
  4. Article - Um conteúdo dentro da página
  5. Footer - O rodapé da página.

Web Scraping


A técnica de web scraping, traduzida como raspagem de dados, é utilizada para extrair dados de páginas web, transformando estruturas web em dados estruturados.
Essas informações podem ser interpredadas de muitas maneiras por exemplo, quantos posts sobre determindado assunto um blog possui, ou especificar por postagens de um autor especifico.
Essa técnica automatiza sistemas e geração de dados para interpretação.

Entendendo SEO e sua utilização num site web


SEO, otimização dos serviços de busca, são varias técnicas aplicadas que garantem a melhora na visibilidade nos serviços de busca, como google.
A partir de diversas métricas, como estrategia, tags, palavras chave e direção do projeto e possível definir qual melhor estratégia para que um site alcance seu público.
